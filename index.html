O que é Git?

Git é um sistema de controle de versão distribuído que permite acompanhar mudanças no código ao longo do tempo.

Ele facilita colaboração, versionamento, histórico e segurança dos arquivos.

GitHub e Repositórios Remotos

GitHub é uma plataforma baseada em Git para hospedar, colaborar e gerenciar projetos.

Comandos Básicos

    git init – inicia um repositório ( Somente quando não for clone )
    git status – mostra alterações
    git add . – adiciona arquivos
    git commit -m "msg" – salva mudanças
    git branch -M main – renomeia a branch principal para main ( usado ao criar um novo repositório )
    git remote add origin https://github.com/user/projeto.git – adiciona um repositório remoto ( usado ao criar um novo repositório )

Editando um Commit

Você pode editar o último commit caso tenha esquecido algum arquivo ou queira alterar a mensagem.

git commit --amend

Para alterar a mensagem diretamente:

git commit --amend -m "Nova mensagem do commit"

Renomear uma Branch

Se você escreveu um nome errado, pode renomear assim:

git branch -m nome-antigo nome-novo

Se estiver na branch atual:

git branch -m nome-novo

Arquivo .gitignore

O arquivo .gitignore serve para ignorar arquivos e pastas no versionamento.

node_modules/ *.env /dist

Após editar o .gitignore, se arquivos já estiverem no Git:

git rm -r --cached . git add . git commit -m "Atualizando gitignore"
